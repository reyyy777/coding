Kelas CSS

Pelajari cara mengatur tampilan halaman web dengan CSS

Pengantar CSS

CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan dan gaya halaman HTML seperti warna, layout, font, dan lainnya.

Selektor CSS

Selektor digunakan untuk memilih elemen HTML yang ingin diberikan gaya. Beberapa contoh selektor:

  • p — memilih semua elemen paragraf
  • .kelas — memilih elemen dengan class tertentu
  • #id — memilih elemen dengan ID tertentu
  • div > p — memilih paragraf langsung di dalam div

Contoh Kode CSS

Berikut contoh dasar penulisan CSS:

body {
    font-family: Arial, sans-serif;
    background-color: #f9f9f9;
    color: #333;
}

h1 {
    color: #0066cc;
    text-align: center;
}

.kotak {
    width: 200px;
    height: 100px;
    background-color: lightblue;
    border: 1px solid #ccc;
}
