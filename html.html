Pengantar HTML

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat dan menyusun halaman web. HTML menggunakan elemen-elemen untuk menyusun konten seperti teks, gambar, link, dan lainnya.

Struktur Dasar HTML

Berikut adalah contoh struktur dasar dari dokumen HTML:

<!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8">
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
    <p>Ini adalah paragraf pertama saya.</p>
  </body>
</html>

Elemen HTML Umum

Beberapa elemen HTML yang sering digunakan:

  • <h1> hingga <h6> – Judul dengan tingkatan
  • <p> – Paragraf teks
  • <a> – Tautan/Link
  • <img> – Gambar
  • <ul>, <ol>, <li> – Daftar
  • <div>, <span> – Pembungkus konten
