Pengantar JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi lebih interaktif. Dengan JavaScript, kita bisa membuat tombol interaktif, validasi form, animasi, dan banyak lagi.

Sintaks Dasar JavaScript

  • let, const — untuk mendeklarasikan variabel
  • if, else — untuk percabangan
  • for, while — untuk perulangan
  • function — untuk membuat fungsi
  • console.log() — untuk menampilkan data di console

Contoh Kode JavaScript

Berikut adalah contoh sederhana JavaScript:

// Menampilkan pesan ke console
console.log("Halo, dunia!");

// Fungsi penjumlahan
function tambah(a, b) {
    return a + b;
}

let hasil = tambah(3, 5);
console.log("Hasil penjumlahan: " + hasil);
